# SysPadaria

Base modular para sistema de padaria com Django, PostgreSQL, Django Templates e Bootstrap.

## Módulos

- Autenticação e usuários
- Vendas práticas / caixa
- Estoque
- Financeiro
- Relatórios e dashboard

## Docker

1. Copie `.env.example` para `.env`.
2. Suba os containers com `docker compose up --build`.
3. Acesse `http://localhost:8000`.

## Próximos passos

1. Criar a venv e instalar dependências.
2. Configurar o `.env`.
3. Rodar as migrações iniciais.
4. Implementar a primeira regra de negócio do módulo escolhido.
